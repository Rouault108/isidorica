## 1. 本文書の範囲

本文書は、理論的基盤から**演繹的に導出**されたサービス設計を定義する。

### 1.1 Isidoricaが提供する概念的空間

Isidoricaは、Libraryを中核とした以下の構造を持つ：

```
┌─────────────────────────────────────────┐
│ Library（知識基盤）【中核】              │
│   - すべてのコンテンツの実体             │
│   - 読者の状態に応じたアダプティブ提供   │
│                                         │
│   └─ Laboratory（体験層）               │
│        記事に埋め込まれる体験要素       │
└─────────────────────────────────────────┘
           ↕ 相互連携
┌─────────────────────────────────────────┐
│ Agora（対話基盤）                        │
│   - Libraryコンテンツへの議論・批判      │
│   - 議論結果がLibraryに還流              │
└─────────────────────────────────────────┘
```

### 1.2 各空間の定義

| 空間 | 定義 | 理論的根拠 | 提供する価値 |
|:---|:---|:---|:---|
| **Library** | 信頼できる知識へのアダプティブなアクセス | 証言の認識論、ZPD、辞書機能論 | 信頼性、適応性、公正性、透明性 |
| **Laboratory** | 能動的推論を通じた学習体験（Library内に埋め込み） | 4E認知、予測処理 | 操作可能性 |
| **Agora** | 批判的相互作用を通じた知識精錬 | 批判的文脈主義、対話的アプローチ | 透明性 |

> **動作詳細**: 各空間の詳細な動作については [PRODUCT_DESIGN.md](./PRODUCT_DESIGN.md) を参照

### 1.3 Libraryのアダプティブ性

Libraryは以下の特性を持つ：

1. **統合されたコンテンツ**: 体系的学習と参照の両方に対応する単一のコンテンツ基盤
2. **アクセスモードの適応**: 読者の目的（P）と時間制約（T）に応じて表示を調整
3. **知識状態への適応**: 読者の知識状態（K）に応じて前提処理・詳細度を調整
4. **学習パス**: キュレーションされた推奨経路と、知識状態に基づく動的調整の組み合わせ

## 2. 対象読者

Isidoricaは**すべての学習者**を対象とする。読者の知識状態、学習スキル、目的、時間制約に関わらず、それぞれの状態に応じてアダプティブに対応する。

### 2.1 対象読者のモデル化

読者を固定的な「タイプ」（例: 初心者、中級者、上級者）で分類するのではなく、**4つの独立した次元**で動的に捉える：

- **K（知識状態）**: 当該トピックについてどれだけ知っているか
- **M（メタ学習スキル）**: 学び方を知っているか
- **P（アクセス目的）**: 何を達成したいか
- **T（時間制約）**: どれだけ時間をかけられるか

> **詳細定義**: 各次元の値、判定タイミング、適応動作は [PRODUCT_DESIGN.md §2](./PRODUCT_DESIGN.md) を参照

### 2.2 教育的公正の原則

4次元モデルは、特に以下の理論的考慮に基づく：

- **文化資本論（Bourdieu）**: 学術的文献に慣れていない読者も排除しない
- **言語コード論（Bernstein）**: 精緻コードに依存しない設計
- **ZPD（Vygotsky）**: 読者の現在地点に合わせた足場かけ

## 3. 提供価値

### 3.1 価値の導出

提供価値は、理論的基盤から以下の論理で導出される：

| 理論的基盤 | 導出される価値 |
|:---|:---|
| 証言の認識論、批判的文脈主義 | 信頼性、透明性 |
| 予測処理、4E認知 | 操作可能性 |
| 文化資本論、言語コード論 | 公正性 |
| ZPD、熟達化逆転効果、辞書機能論 | 適応性 |

### 3.2 読者への価値（5価値）

| 価値 | 定義 | 対応する設計原則 |
|:---|:---|:---|
| **信頼性（Trustworthiness）** | 全記述に出典を紐付け、読者が自ら検証できる | E-1（形成過程可視化）, E-3（認識的信念涵養） |
| **操作可能性（Interactivity）** | 概念を操作・介入を通じて体験できる | B-1（先行知識活性化）, B-2（身体接地）, D-1（生産的混乱設計） |
| **公正性（Equity）** | 文化資本の有無に関わらず学べる | C-1（暗黙知明示化）, C-2（外的構造）, C-3（学習意味明示）, C-4（情報等価性） |
| **透明性（Transparency）** | 知識形成の批判・改訂プロセスが公開される | E-1（形成過程可視化）, E-2（認識的多様性） |
| **適応性（Adaptivity）** | 読者の状態（K, M, P, T）に応じて情報提供を動的に調整する | §2 4次元モデル, KNOWLEDGE_STATE_MODEL, METALEARNING_DESIGN |

> **信頼性 vs 透明性**: 信頼性は「検証可能性（出典）」、透明性は「プロセスの可視性（批判・改訂）」を指す。

## 5. サービスの射程と限界

> 本セクションは、Isidoricaが**解決できる問題**と**解決できない問題**を明確に区別する。教育的公正の実現は、教育プラットフォーム単独では不可能であり、サービスの限界を正直に認識することが重要である。

### 5.1 解決可能な領域

| 領域 | 説明 | 根拠理論 |
|------|------|----------|
| **アクセス後の学習体験** | 高品質な構造化コンテンツ、精密コードの明示化 | CLT, バーンスタイン |
| **経済的障壁の除去** | 無料でアクセス可能なコンテンツ | — |
| **暗黙知の明示化** | 「問い」の共有、文脈の提示、精密コードの教授 | ブルデュー, ガダマー |
| **意欲の維持・強化** | 驚き→好奇心の変換、有能感フィードバック、構造の提供 | SDT, CVT, 苅谷 |
| **メタ認知の支援** | JOL較正、流暢性の錯覚への対抗 | Nelson & Narens |

### 5.2 解決不可能な領域

| 領域 | 説明 | 根拠理論 |
|------|------|----------|
| **デジタルデバイド** | インターネットにアクセスできない層には届かない | デジタル格差論 |
| **意欲格差の根本原因** | 「学ぶ意味」が内面化されていない家庭の子どもは「来ない」 | 苅谷、ブルデュー |
| **時間的余裕の欠如** | 働かなければならない学生、家事負担の大きい層 | 時間貧困論 |
| **親の関与とサポート** | 家庭の教育的身構えの格差 | ペアレントクラシー、苅谷 |
| **物理的学習環境** | 静かな学習スペースがない、デバイスがない | ケイパビリティ論 |
| **精神的健康** | メンタルヘルスの問題が学習を妨げる | 心理学 |
| **社会的スティグマ** | 「勉強すること」自体が否定される環境 | 社会心理学 |

### 5.3 サービスの位置づけ

Isidoricaは教育的公正にとって**必要条件（Necessary Condition）**であるが、**十分条件（Sufficient Condition）**ではない。

- **必要条件**: アクセスした人に対して、文化資本や言語コードの格差を縮小する最高の学習体験を提供する。このような質の高いコンテンツが無料で存在しなければ、格差縮小の可能性すらない。
- **十分条件ではない**: §5.2に示した領域はサービス単独では解決できず、社会的・制度的介入が不可欠である。

##### 資格主義批判と非資格発行方針

Isidoricaは意図的に**資格・証明書を発行しない**。この方針は **Randall Collins** (*The Credential Society*, 1979) の資格主義（Credentialism）批判に理論的根拠を持つ。

Collinsは、学歴インフレーション（credential inflation）——必要以上の学歴が求められる現象——を批判し、教育が「スキルの獲得」ではなく「地位競争の道具」になっていると論じた。Isidoricaは、学習を資格取得のための手段ではなく、**知識そのものの獲得**という本来の目的に焦点を当てる。
