## 1. 著作権ガイダンス

本セクションでは、Isidoricaにおける著作物の引用および利用に関するガイドラインを定める。

### 1.1 適用法令と基本方針

Isidoricaはグローバルにアクセス可能なサービスであるため、以下の主要な法域における著作権法に準拠する。

| 法域 | 法令 | 引用に関する主な規定 |
| :--- | :--- | :--- |
| **日本** | 著作権法 | 第32条（引用）、第35条（学校その他の教育機関における複製等） |
| **米国** | Copyright Act | Section 107（Fair Use）、Section 110（教育目的の例外） |
| **EU** | 著作権指令 | InfoSoc Directive 2001/29/EC Article 5(3)(d)（引用例外）、CDSM Directive 2019/790 Article 5（教育目的） |

**基本方針**: Isidoricaは、各法域の法的要件のうち**最も厳格なものに準拠する**ことで、どの法域からアクセスされても法的リスクを最小化する。

### 1.2 テキストの引用

#### 1.2.1 法域別の引用要件

##### 日本（著作権法第32条）

| 要件 | 説明 |
| :--- | :--- |
| **公表された著作物** | 未公表の著作物は引用不可 |
| **公正な慣行に合致** | 学術・批評・報道等の正当な目的 |
| **引用の目的上正当な範囲内** | 必要最小限の分量 |
| **出所の明示** | 著作者名、著作物名、出典ページ等 |
| **主従関係** | 引用部分が「従」、自己の著作物が「主」 |
| **明瞭区別性** | 引用部分が明確に区別されていること |

##### 米国（Fair Use - Section 107）

Fair Useは以下の**4要素テスト**によりケースバイケースで判断される。

| 要素 | Isidoricaへの適用 |
| :--- | :--- |
| **1. 使用の目的と性質** | 教育・批評・解説目的、非営利、変容的使用（transformative）であれば有利 |
| **2. 著作物の性質** | 事実的著作物（学術論文等）の方が創作的著作物（小説等）より有利 |
| **3. 使用量と実質性** | 作品全体に対する引用部分の割合、「心臓部」の使用を避ける |
| **4. 市場への影響** | オリジナル作品の代替とならないこと |

> **注意**: 教育目的であってもFair Useが自動的に認められるわけではない。

##### EU（InfoSoc Directive Article 5(3)(d)）

| 要件 | 説明 |
| :--- | :--- |
| **合法的に公衆に提供された著作物** | 合法的に公開されていること |
| **批評またはレビュー目的** | 批評、レビュー、解説等の正当な目的 |
| **公正な慣行に準拠** | 目的に必要な範囲内 |
| **出典の明示** | 著作者名を含む出典の明示（不可能な場合を除く） |

#### 1.2.2 Isidoricaにおける引用の共通原則

上記の法域要件を満たすため、Isidoricaでは以下の**共通原則**を適用する。

| 原則 | 説明 | 例 |
| :--- | :--- | :--- |
| **1. 必要最小限** | 引用は議論・解説に必要な範囲に限定 | 全文引用ではなく、関連する段落のみ |
| **2. 主従関係の維持** | 記事本文が「主」、引用が「従」 | 引用量が記事全体の大部分を占めない |
| **3. 出典の完全な明示** | 著作者、著作物名、出版年、ページ番号を明記 | `[@tanaka-game-theory-2020, p. 42]` |
| **4. 明瞭な区別** | 引用部分を明確に区別 | `>` blockquoteまたは`::quote`ディレクティブ |
| **5. 変容的価値の付加** | 単なる複製ではなく、分析・批評・解説を加える | 引用後に解説文を必ず付加 |
| **6. 市場への影響回避** | オリジナル作品の代替とならない | 教科書の全章を引用するのではなく、要点のみ |

#### 1.2.3 引用の書式

Isidoricaでは**Pandoc標準の引用記法**を採用する。詳細は [GOVERNANCE.md](./GOVERNANCE.md) の「引用記法」セクションを参照。

```markdown
<!-- 短い引用（1文程度） -->
アリストテレスは「人間は本性上、知ることを欲する」[@aristotle-metaphysics-ja, p. 980a]と述べた。

<!-- 長い引用（blockquote） -->
> すべての人間は、生まれつき知ることを欲する。
> その証拠に、感覚的知覚への愛着がある。
> 
> — [@aristotle-metaphysics-ja, p. 980a]

引用元の解説: アリストテレスはここで...（解説を必ず付加）
```

| 記法 | 意味 | 例 |
| :--- | :--- | :--- |
| `[@source-id]` | 出典全体を参照 | `[@heidegger-being-time-1962]` |
| `[@source-id, p. 45]` | 特定ページを参照 | `[@heidegger-being-time-ja, p. 45]` |
| `[@source-id, pp. 45-50]` | ページ範囲を参照 | `[@heidegger-being-time-ja, pp. 45-50]` |
| `[@id-ja, p. 45; orig. p. 42]` | 翻訳版と原典ページを併記 | 日本語版p.45、原典p.42 |

### 1.3 画像・図表の引用

画像・図表は**テキストの引用よりも厳しい制限**がある。

#### 1.3.1 法域別の規定

| 法域 | 画像引用の可否 | 要件 |
| :--- | :--- | :--- |
| **日本** | 制限的に可能 | 鑑賞目的ではなく、解説・批評に必要な範囲で、出典明示 |
| **米国** | Fair Use判断 | 4要素テストで個別判断、サムネイルは比較的認められやすい |
| **EU** | 制限的に可能 | 批評・レビュー目的、公正な慣行に準拠、出典明示 |

#### 1.3.2 Isidoricaにおける画像利用の分類

| 分類 | 許諾要否 | 対応 |
| :--- | :--- | :--- |
| **自作** | 不要 | 執筆者が作成した図表・イラスト |
| **パブリックドメイン** | 不要 | 著作権が消滅した作品（著作者死後70年等） |
| **CC0** | 不要 | 著作権放棄された作品 |
| **CC BY / CC BY-SA等** | 不要（条件遵守） | ライセンス条件（帰属表示等）に従う |
| **著作権あり（Fair Use主張）** | 限定的に可能 | 批評・解説目的で必要不可欠な場合のみ、最小限のサイズで使用 |
| **著作権あり（許諾取得）** | 許諾要 | 権利者から書面による許諾を取得 |

#### 1.3.3 画像引用のベストプラクティス

| 原則 | 説明 |
| :--- | :--- |
| **1. 許諾不要の素材を優先** | CC0、パブリックドメイン、自作を第一選択肢とする |
| **2. 必要不可欠性の証明** | その画像がなければ説明が成立しない場合のみ引用 |
| **3. 最小限のサイズ** | 原寸大ではなく、説明に必要な解像度・サイズで使用 |
| **4. 出典の完全明示** | 著作者名、作品名、所蔵先、ライセンス情報を明記 |
| **5. 批評・解説の付加** | 画像自体を鑑賞させるのではなく、必ず分析・解説を付加 |

### 1.4 コード引用

プログラムコードの引用については、`GOVERNANCE.md` の「コードスニペットの引用ポリシー」を参照。

### 1.5 許諾の取得と記録

#### 1.5.1 許諾取得のフロー

| ステップ | 担当 | 内容 |
| :--- | :--- | :--- |
| **1. 許諾要否の判断** | 執筆者 | 上記分類に基づき許諾が必要か判断 |
| **2. 許諾申請** | 執筆者 | 権利者に対して使用許諾を申請 |
| **3. 許諾の取得** | 執筆者 | 書面（メール可）で許諾を取得 |
| **4. 証拠の提出** | 執筆者 | PRに許諾証拠を添付 |
| **5. 確認** | レビュワー | 証拠を確認し、PRをマージ |

#### 1.5.2 許諾記録の保存

| 項目 | 保存場所 | 保持期間 |
| :--- | :--- | :--- |
| **許諾メール/書面** | 非公開リポジトリまたはセキュアストレージ | 著作権保護期間終了まで |
| **メタデータ** | `bibliography.json` 内の該当エントリ | 同上 |

### 1.6 引用違反への対応

Isidoricaの記事において引用ルールに違反している可能性がある場合:

| シナリオ | 対応 |
| :--- | :--- |
| **執筆者自身が気づいた場合** | 速やかに修正PRを作成、または問題の範囲が大きい場合は記事を非公開化 |
| **レビュー時に発見された場合** | PRを承認せず、修正を依頼 |
| **公開後に第三者から指摘された場合** | `5. 著作権侵害の申し立て対応` のフローに従う |

## 2. プライバシーポリシー

本セクションでは、Isidoricaにおける個人情報およびプライバシーの取り扱い方針を定める。

### 2.1 基本方針

Isidoricaは、利用者のプライバシーを最大限に尊重し、**必要最小限のデータのみを収集する**方針を採用する。

| 原則 | 説明 |
| :--- | :--- |
| **データ最小化** | サービス運営に必要不可欠なデータのみを収集 |
| **目的限定** | 収集したデータは明示した目的のみに使用 |
| **透明性** | 何を収集し、どう使用するかを明確に開示 |
| **セキュリティ** | 収集したデータを適切に保護 |
| **サードパーティ制限** | プライバシーを侵害する可能性のあるサードパーティツールは使用しない |

### 2.2 適用法令

| 法域 | 法令 |
| :--- | :--- |
| **日本** | 個人情報の保護に関する法律（個人情報保護法） |
| **EU** | General Data Protection Regulation（GDPR） |
| **米国カリフォルニア州** | California Consumer Privacy Act（CCPA） |

### 2.3 収集するデータ

#### 2.3.1 アクセス解析データ（Cloudflare Web Analytics）

| 項目 | 収集 | 説明 |
| :--- | :--- | :--- |
| **ページビュー** | ✅ | どのページが何回閲覧されたか |
| **滞在時間** | ✅ | ページの閲覧時間 |
| **リファラー** | ✅ | 参照元URL（どこから来たか） |
| **Core Web Vitals** | ✅ | LCP, FID, CLS等のパフォーマンス指標 |
| **IPアドレス** | ❌ | **収集しない**（Cloudflareが匿名化処理） |
| **Cookie** | ❌ | **使用しない** |
| **ユーザー識別子** | ❌ | **収集しない** |

**利用目的**: サイトのパフォーマンス改善、コンテンツ改善のための統計分析

**法的根拠（GDPR）**: 正当な利益（Legitimate Interest）- ウェブサイト運営の改善

> **注**: Cloudflare Web AnalyticsはCookieを使用せず、IPアドレスを保存しないため、**GDPR/CCPAに準拠**し、Cookie同意バナーは不要。

#### 2.3.2 検索アナリティクス（Algolia Analytics）

| 項目 | 収集 | 説明 |
| :--- | :--- | :--- |
| **検索クエリ** | ✅ | 入力された検索キーワード |
| **クリック率** | ✅ | 検索結果のどの項目がクリックされたか |
| **ゼロヒット率** | ✅ | 結果が0件だった検索の割合 |
| **IPアドレス** | ❌ | **収集しない**（Algolia側で匿名化設定） |
| **ユーザー識別子** | ❌ | **収集しない** |

**利用目的**: 検索品質の改善（人気クエリの把握、ゼロヒット率の削減）

**利用制限**: 検索品質改善以外の目的での利用は認めない

**法的根拠（GDPR）**: 正当な利益（Legitimate Interest）- 検索機能の改善

#### 2.3.3 お問い合わせフォーム

| 項目 | 収集条件 | 説明 |
| :--- | :--- | :--- |
| **氏名** | 目的達成に必要な場合のみ | 著作権侵害申し立て等、法的要件がある場合 |
| **メールアドレス** | 返信が必要な場合のみ | 返信以外の目的には使用しない |
| **問い合わせ内容** | ✅（必須） | お問い合わせへの対応のため |

**利用目的**: お問い合わせへの対応のみ

**保持期間**: 対応完了後5年（法的要件への対応のため）

**法的根拠（GDPR）**: 契約履行/法的義務の遵守

#### 2.3.4 Cookieの使用

| Cookie種別 | 使用 | 説明 |
| :--- | :--- | :--- |
| **必須Cookie** | ⚪ 最小限 | セッション管理、CSRF保護等、サービス提供に不可欠なもののみ |
| **分析Cookie** | ❌ | 使用しない（Cloudflare AnalyticsはCookie不使用） |
| **広告Cookie** | ❌ | 使用しない |
| **トラッキングCookie** | ❌ | 使用しない |

**方針**: 必須Cookie以外は一切使用しない。必須Cookieのみの使用であるため、**Cookie同意バナーは不要**。

#### 2.3.5 ユーザーアカウント

> **注**: Isidoricaはアカウント機能を備えておらず、個人を特定する認証情報は収集しない。

### 2.4 収集しないデータ

Isidoricaは以下のデータを**一切収集しない**。

| 項目 | 説明 |
| :--- | :--- |
| **IPアドレス**（長期保存） | Cloudflare/Algoliaレベルで匿名化 |
| **位置情報** | GPS、IPジオロケーション等 |
| **デバイスフィンガープリント** | ブラウザ/デバイスを特定する情報 |
| **閲覧履歴**（個人単位） | 個人を特定した閲覧行動の追跡 |
| **センシティブ情報** | 人種、宗教、政治的見解、健康情報等 |

### 2.5 禁止されるサードパーティツール

以下のサードパーティツールの使用は**禁止**する。

| ツール | 禁止理由 |
| :--- | :--- |
| **Google Analytics** | ユーザートラッキング、Googleへのデータ送信 |
| **Facebook Pixel** | 広告トラッキング |
| **その他の広告/トラッキングツール** | プライバシー侵害のリスク |

**許可されるツール**: Cloudflare Web Analytics、Algolia（匿名化設定必須）のみ

### 2.6 データの第三者提供

| 提供先 | 条件 |
| :--- | :--- |
| **Cloudflare** | アクセス解析・CDNサービス提供のため（匿名化データのみ） |
| **Algolia** | 検索サービス提供のため（匿名化データのみ） |
| **法執行機関** | 法的義務がある場合のみ（裁判所命令等） |
| **その他の第三者** | ❌ 提供しない |

### 2.7 データ保持期間

| データ種別 | 保持期間 | 根拠 |
| :--- | :--- | :--- |
| **アクセス解析データ** | Cloudflareの標準期間（最大6ヶ月） | 統計分析に必要な最小期間 |
| **検索アナリティクス** | Algoliaの標準期間（最大90日） | 検索品質改善に必要な最小期間 |
| **お問い合わせデータ** | 対応完了後5年 | 法的要件への対応 |
| **ユーザーアカウント** | アカウント削除まで | ユーザーの要求に応じて削除 |

### 2.10 Cookie同意バナーについて

Isidoricaは**Cookie同意バナーを表示しない**方針である。

**理由**:
- 分析・追跡目的のCookieを使用しない
- 必須Cookie（セッション管理、CSRF保護）のみの使用
- GDPR/ePrivacy指令において、必須Cookieは同意不要

### 2.11 子どものプライバシー

Isidoricaは全年齢向けの教育コンテンツを提供し、**子どもを含む全ての利用者がアカウント作成なしで全機能を利用できる**よう設計されている。

| 項目 | 内容 |
| :--- | :--- |
| **閲覧・学習** | アカウント不要で全機能利用可能 |
| **学習進捗** | ブラウザのLocalStorageに保存（個人情報を収集しない） |
| **アカウント** | 13歳以上のみ。デバイス間同期が必要な場合のオプション機能 |

**13歳未満の子どもからは個人情報を収集しない**（COPPA準拠）。

13歳未満の利用者がデバイス間で進捗を同期したい場合は、保護者のアカウントを利用するか、LocalStorageのバックアップ機能（将来実装予定）を使用することを推奨する。

## 3. 利用規約

本利用規約（以下「本規約」）は、Isidorica（以下「本サービス」）の利用条件を定めるものである。本サービスを利用することにより、利用者は本規約に同意したものとみなされる。

### 3.1 サービスの概要

| 項目 | 内容 |
| :--- | :--- |
| **サービス名** | Isidorica |
| **運営者** | Isidorica 運営チーム |
| **サービス内容** | 学術・教育コンテンツの提供、インタラクティブな学習体験（Laboratory）の提供 |

### 5.5 異議申立て（Counter-Notice）手続き

発信者（執筆者）は、削除されたコンテンツについて異議を申し立てることができる。

#### 5.5.1 異議申立ての要件

| 必須項目 | 説明 |
| :--- | :--- |
| **1. 署名** | 発信者の電子署名または物理署名 |
| **2. 削除コンテンツの特定** | 削除されたコンテンツの明確な特定 |
| **3. 誠実な信念の表明** | 誤りまたは誤認により削除されたという誠実な信念の表明 |
| **4. 管轄への同意** | 該当法域の管轄裁判所への同意（DMCA） |
| **5. 連絡先** | 発信者の氏名、住所、連絡先 |

#### 5.5.2 異議申立て後のフロー

| ステップ | 期限 | 対応 |
| :--- | :--- | :--- |
| **1. 異議申立て受領** | - | 申告者（著作権者）へ異議申立て内容を転送 |
| **2. 申告者の対応期限（DMCA）** | 10〜14営業日 | 申告者が法的措置を講じない場合、コンテンツを復元 |
| **3. 法的措置の通知** | - | 申告者が訴訟を提起した場合、コンテンツは非公開のまま維持 |

### 5.6 リピートインフリンジャーポリシー

DMCAセーフハーバーの要件として、繰り返し著作権侵害を行うユーザーへの対応方針を定める。

| 違反回数 | 対応 |
| :--- | :--- |
| **1回目** | 警告通知、該当コンテンツの削除 |
| **2回目** | 厳重警告、該当コンテンツの削除、一定期間の投稿制限 |
| **3回目以上** | アカウント停止または永久BAN |

> **注**: 現時点ではユーザーアカウント機能は未実装のため、本ポリシーはGitHubアカウント単位で適用する。

## 6. 盗作・剽窃への対応 (Plagiarism & Copyright Enforcement)

本セクションでは、**Isidoricaのコンテンツが外部で無断使用・盗作された場合**の対応方針を定める。

> **注**: セクション5は「Isidoricaが侵害者として申し立てを受けた場合」を扱う。本セクションは「Isidoricaが権利者として対応する場合」を扱う。

### 6.1 基本方針

#### 6.1.1 オープンライセンスとの関係

Isidoricaのコンテンツはオープンライセンス（CC BY-SA 4.0）で公開されているため、**複製・改変・再配布自体は歓迎される**。

| 行為 | 許可 | 条件 |
| :--- | :--- | :--- |
| **複製** | ✅ 許可 | 帰属表示が必要 |
| **改変** | ✅ 許可 | 帰属表示 + 同一ライセンスで公開 |
| **商用利用** | ✅ 許可 | 帰属表示 + 同一ライセンスで公開 |
| **翻訳** | ✅ 許可 | 帰属表示 + 同一ライセンスで公開 |

#### 6.1.2 問題となるケース

以下のケースは**ライセンス違反**または**著作権侵害**として対応する。

| 侵害の種類 | 説明 | 深刻度 |
| :--- | :--- | :--- |
| **帰属表示の欠落** | Isidoricaを出典として明記していない | 低〜中 |
| **ライセンス未記載** | CC BY-SA 4.0のライセンス表示がない | 低〜中 |
| **異なるライセンスでの再配布** | 「All Rights Reserved」等で公開 | 中〜高 |
| **「自作」と偽る** | 自分が作成したコンテンツと主張 | 高 |
| **誤情報の付加** | 内容を改ざんし誤った情報として拡散 | 高 |

#### 6.1.3 Isidoricaの立ち位置

| 原則 | 説明 |
| :--- | :--- |
| **寛容さ** | オープンライセンスの精神を尊重し、過度に攻撃的にならない |
| **教育的アプローチ** | ライセンスの意味を知らない人も多い、まず説明から |
| **証拠主義** | 感情ではなく事実に基づいて判断 |
| **比例原則** | 侵害の深刻さに比例した対応 |
| **リソース効率** | 限られた時間を効果的に使う |

### 6.2 侵害の検出

#### 6.2.1 検出方法

| 方法 | 頻度 | 対象 | 備考 |

### 6.6 法的措置

#### 6.6.1 DMCA通知の発行（Isidoricaが権利者として）

米国でホスティングされているサイトに対しては、DMCA Takedown Noticeを発行できる。

**DMCA通知に含める項目:**

| 項目 | 説明 |
| :--- | :--- |
| **1. 署名** | Isidorica運営責任者の電子署名 |
| **2. 著作物の特定** | 侵害されているIsidorica記事のURL |
| **3. 侵害コンテンツの特定** | 侵害者サイトのURL |
| **4. 連絡先** | copyright@isidorica.org |
| **5. 誠実な信念の表明** | 当該使用が許可されていない旨 |
| **6. 正確性の宣誓** | 偽証罪の制裁のもとで正確である旨 |

#### 6.6.2 ホスティング会社への通報

| ホスティング | Abuse連絡先 |
| :--- | :--- |
| **Cloudflare** | https://www.cloudflare.com/abuse/ |
| **AWS** | https://aws.amazon.com/forms/report-abuse |
| **Google Cloud** | https://support.google.com/code/contact/cloud_platform_report |
| **その他** | Whois情報から特定 |

#### 6.6.3 法的措置を取る/取らない判断基準

**法的措置を検討するケース:**
- 商用サイトによる大規模な無断転載
- 警告後も意図的に侵害を継続
- Isidoricaの信頼性を損なう可能性（誤情報の付加等）
- 他の被害者がいる可能性

**法的措置を取らないケース:**
- 個人ブログでの軽微な引用
- 教育機関での使用（帰属表示依頼のみ）
- 連絡先が不明で対応不可能
- 対応コストが損害を上回る

| **結果** | 解決 / 未解決 / 対応不要 |
