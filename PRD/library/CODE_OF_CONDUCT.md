# 行動規範 (Code of Conduct)

本ドキュメントでは、Isidoricaコミュニティにおける行動規範を定義する。
本規範は、単なるマナー集ではなく、Isidoricaの核心的価値である「教育的公正」と「認識的公平性」を守るための**社会契約（Social Contract）**である。

---

## 1. 目的と理念

我々の目的は、あらゆる背景を持つ人々が、その属性（人種、性別、**知識レベル**、**文化的背景**等）に関わらず、対等な立場で知識の探究に参加できる環境を保障することである。

### 1.1 認識的安全性 (Epistemic Safety)

Isidoricaは「心理的安全性」を超えた**「認識的安全性」**を保障する。
これは、**「無知であること」「間違えること」「問うこと」が、嘲笑や排除の対象にならず、むしろ知識生産への貢献として歓迎される状態**を指す。

| 概念 | 定義 |
| :--- | :--- |
| **出典至上主義** | 議論の勝敗は「誰が言ったか（権威）」ではなく「何に基づいているか（出典）」で決まる。 |
| **文化資本の濫用防止** | 構造的な「知の非対称性」を、対人関係における優位性維持のために利用してはならない。 |

### 1.2 開かれた参加 (Open Participation)

Isidoricaは**Linus's Law**（「目が十分に多ければ、全てのバグは浅い」— Eric S. Raymond, *The Cathedral and the Bazaar*, 1999）を採用する。品質は個人の専門性ではなく、**出典の検証可能性（Source Supremacy）**と**レビュー**によって担保される。

| 原則 | 内容 |
| :--- | :--- |
| **資格不問** | 貢献に専門資格や学歴は不要である。誰もが貢献でき、誰もがレビューできる |
| **能力より姿勢** | 求められるのは専門的能力ではなく、誠実な姿勢（§2.3参照） |
| **集団的品質保証** | 個人の専門性に依存せず、レビュープロセスと出典検証で品質を担保 |

## 2. 推奨される行動 (Our Standards)

我々は、協調的で包括的なコミュニティを形成するために、以下の行動を推奨する。

### 2.1 経験とつまずきの共有 (Sharing Struggles)
学習理論に基づけば、「つまずき」は恥ずべき失敗ではなく、教材の不備を発見するための重要な情報である。

*   **推奨**: 「この概念の定義が理解できませんでした」と率直に伝える。
*   **推奨**: 専門用語を使わず、誰もが理解できる言葉で説明を試みる（明示化）。

### 2.2 建設的な相互批判 (Critical Contextualism)
批判は個人ではなく「主張」に対して向けられるべきである（[CORE_FOUNDATIONS.md §2.7](../shared/CORE_FOUNDATIONS.md) 参照）。

*   **推奨**: 「あなたは間違っている」ではなく、「出典Aによれば別の解釈も可能です」と提案する。
*   **推奨**: 相手の背景知識を尊重し、善意の解釈（Principle of Charity）をもって対話する。

### 2.3 誠実性 (Integrity)

貢献者は以下の誠実性の規範を遵守する。これは専門的能力の要求ではなく、**姿勢**の期待である。

| 原則 | 内容 |
| :--- | :--- |
| **出典の明示** | 主張にはすべて検証可能な出典を付ける。「〜である」という断言には根拠を示す |
| **不確実性の開示** | 確信がない場合はその旨を明示する（「〜と思われる」「要検証」等）。知らないことを知っていると言わない |
| **認識的謙虚** | 自分が間違っている可能性を常に認め、訂正を受け入れる姿勢を持つ |
| **利益相反の開示** | 経済的・思想的・個人的利害関係がある場合は自発的に開示する |

## 3. 禁止される行動 (Unacceptable Behaviors)

Isidoricaは、以下のハラスメント行為を一切容認しない。

### 3.1 知的ハラスメント (Intellectual Harassment)

知識や経験の差を利用して、他者を排除、萎縮、あるいは嘲笑する行為。
ブルデューが指摘した「象徴暴力（Symbolic Violence）」は社会構造的な現象であるが、Isidoricaコミュニティにおいては、この**構造的な非対称性を個人が濫用する行為**を重大な倫理違反と定義する。

| 分類 | 具体例（違反行為） | 理由 |
| :--- | :--- | :--- |
| **知識によるマウンティング** | 「こんな基本的なことも知らないのか」「常識だ」 | 初学者の学習意欲を削ぎ、階層構造（権威主義）を作るため。 |
| **衒学的排除** | 必要以上に難解な専門用語を使い、相手を煙に巻く | コミュニケーションを阻害し、文化資本による排除を行うため。 |
| **問いの封殺** | 「ググレカス（ggrks）」「過去ログ読め」 | 問いを発すること自体へのコストを高め、認識的安全性を損なうため。 |
| **シーライオニング** | 誠実な議論を装い、執拗な質問で相手を消耗させる | 対話の建設性を損なうため。 |

### 3.2 一般的なハラスメント

*   性別、性的指向、人種、宗教、障害、身体的特徴、年齢、社会的地位などに基づく攻撃的・差別的な言動。
*   個人の同意のない私的情報の公開（ドキシング）。
*   脅迫、ストーキング、暴力の示唆。

## 4. 執行プロセス (Enforcement)

コミュニティリーダー（Core Maintainerおよび指名されたモデレーター）は、本規範に違反する行動に対し、是正措置を講じる責任と権限を持つ。

### 4.1 報告
違反を目撃した場合、または被害を受けた場合は、直ちに以下の窓口へ報告すること。
*   **GitHub Issue**: `conduct` ラベルを使用（公開/非公開選択可）
*   **Direct Message**: Core Maintainerへ直接連絡
*   **Email**: `conduct@isidorica.org`（仮）

**匿名性**: 報告者のプライバシーは厳守され、違反者に対し報告者が特定されることはない。

### 4.2 処分レベル
違反の深刻度と頻度に応じ、以下の処分が適用される。

1.  **是正（Correction）**:
    *   **内容**: コミュニティへの悪影響を指摘し、当該言動の削除・修正を求める。
    *   **対象**: 軽微な、あるいは過失による違反。
2.  **警告（Warning）**:
    *   **内容**: 公式な警告を行い、再発時の処分を通告する。
    *   **対象**: 明確なルール違反、または是正要請の無視。
3.  **一時停止（Temporary Ban）**:
    *   **内容**: 一定期間（1週間〜6ヶ月）、コミュニティからの関与（コメント、PR等）を禁止する。
    *   **対象**: 深刻なハラスメント、または繰り返される違反。
4.  **永久追放（Permanent Ban）**:
    *   **内容**: コミュニティからの永久的な排除。
    *   **対象**: 脅迫、暴力、ドキシング、または改善の見込みがない継続的な知的ハラスメント。

> **BAN逃れ (Ban Evasion)**:
> 処分を受けた個人が、別のアカウントを作成してコミュニティに参加しようとする行為は、それ自体が重大な違反とみなされ、発見次第、即時の永久追放対象となる。

## 5. 適用範囲 (Scope)

本行動規範は、Isidoricaプロジェクトに関連する全ての空間に適用される。

*   GitHub リポジトリ（Issue, PR, Wiki, Discussions）
*   公式チャット/フォーラム（Discord等）
*   プロジェクトを代表して行われる公的な場（カンファレンス、SNS等）

個人のSNSアカウント等における発言であっても、Isidoricaコミュニティへの明確な攻撃や、参加者へのドキシングが含まれる場合は、本規範の対象となりうる。
